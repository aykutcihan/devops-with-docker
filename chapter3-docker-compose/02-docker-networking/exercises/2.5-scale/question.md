# Exercise 2.5 - Scale

Clone the project: https://github.com/docker-hy/material-applications/tree/main/scaling-exercise

The project already includes a `docker-compose.yml`. Start it with `docker compose up`.

The application should be accessible at http://localhost:3000. A load balancer is included for scaling. Scale the compute containers so that the button in the application turns green.

Submit the command used to run the containers.
