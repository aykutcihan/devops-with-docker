# Exercise 2.1 - Getting started

Use `devopsdockeruh/simple-web-service` (linux/amd64) as the image for a new service in a `docker-compose.yaml` file.

Set the container to output logs into a file named `text.log`. You should be able to make it work without extra permissions. Make sure the log file gets created and information is written there by following the logs.

Submit the `docker-compose.yaml`.
