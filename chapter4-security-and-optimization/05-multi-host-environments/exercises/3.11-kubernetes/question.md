# Exercise 3.11 - Kubernetes

Familiarize yourself with Kubernetes terminology and draw a diagram describing what "parts" Kubernetes contains and how they relate to each other.

## Requirements

- At least **3 host machines** in the Kubernetes cluster
- Cluster is running **2 applications** (e.g. a game server and a blog)
- Include **"your own computer"** sending instructions via `kubectl`
- Include an **HTTP request from the internet** reaching an application
- Label at least **4 of**: Pod, Cluster, Container, Service, Volume

## Tools

- draw.io (recommended): https://draw.io
- Any other diagramming tool

## Answer

Submit the URL where the diagram can be found (must be publicly accessible).
